Setup Postman
1
Create Environment
រក្សាទុក base_url និង token
▼
  1. Postman → Environments → + Add
  2. Name: Rous Cheat Local
  3. បន្ថែម variables:
VariableInitial ValueNote
base_urlhttp://localhost:8000/api/v1API root
token(leave empty)Auto-fill after login
recipe_id1Quick test
2
Set Collection Authorization
Protected requests inherit ស្វ័យប្រវត្តិ
▼
  1. Click collection → Edit → tab Authorization
  2. Type → Bearer Token
  3. Token → {{token}}
  4. Save
💡
Protected endpoint → Inherit auth from parent. Public endpoint → No Auth.

Auth — Register
POST
Register new user
ត្រឡប់ JWT token
{{base_url}}/auth/register
📋
Body → raw → JSON
{ "name": "Dara Sok", "email": "dara@example.com", "password": "password123" }
{ "success": true, "data": { "user": { "id": 1, "name": "Dara Sok", "email": "dara@example.com" }, "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..." } }
201Created — user + token
409Email already registered
422Validation failed
🤖
Paste ក្នុង Tests tab — auto-save token ក្រោយ register/login
// Auto-save JWT to environment const res = pm.response.json(); if (res.success && res.data.token) { pm.environment.set("token", res.data.token); console.log("✅ Token saved!"); }
